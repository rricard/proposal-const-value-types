ECMAScript Language Types

The Record Type

The Record type is a coumpound primitive type. It is an ordered collection of primitive data properties.

A primitive data property associates a String key value with a primitive value or a coumpound primitive value called associated value.

Properties are identified using String key values. A String property key value is strictly an ECMAScript String value.

The collection is ordered (TODO: define order used) using the String key values regardless of the associated value. Therefore, there can't be more than one associated value for a given String key.

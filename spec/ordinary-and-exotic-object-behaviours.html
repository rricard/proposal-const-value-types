Ordinary and Exotic Objects Behaviours

Built-in Exotic Object Internal Methods and Slots

Record Exotic Objects

A Record object is an exotic object that encapsuates a Record value and exposes virtual string-indexed data properties corresponding to the individual keys set on the underlying Record value. All keys properties are non-writable and non-configurable.

An object is a Record exotic object (or simply, a Record object) if its following internal methods use the following implementations. These methods are installed in RecordCreate.

Record exotic objects have the same internal slots as ordinary objects. They also have a [[RecordData]] internal slot.

[[SetPrototypeOf]] ( _V_ )

When the [[SetPrototypeOf]] internal method of a Record exotic object _R_ is called with argument _V_, the following steps are taken:

1. Return ! SetImmutablePrototype(_R_, _V_).

[[IsExtensible]] ()

When the [[IsExtensible]] internal method of a Record exotic object _R_ is called, the following steps are taken:

1. Return *false*.

[[GetOwnProperty]] ( _P_ )

When the [[GetOwnProperty]] internal method of a Record exotic object _R_ is called with property key _P_, the following steps are taken:

1. Assert: IsPropertyKey(_P_) is *true*. 1. If Type(_P_) is a Symbol, return *undefined*. 1. Let _getResult_ be ! RecordGet(_R_, _P_). 1. If _getResult_ is *undefined*, return *undefined*. 1. Return the PropertyDescriptor { [[Value]]: _getResult_, [[Writable]]: *false*, [[Enumerable]]: *true*, [[Configurable]]: *false* }.

[[DefineOwnProperty]] ( _P_, _Desc_ )

When the [[DefineOwnProperty]] internal method of a Record exotic object _R_ is called with property key _P_, and Property Descriptor _Desc_ the following steps are taken:

1. Assert: IsPropertyKey(_P_) is *true*. 1. Let _current_ be ! RecordGet(_R_, _P_). 1. If _current_ is *undefined* return *false*. 1. Assert: IsDataDescriptor(_Desc_) is *true*. 1. If _Desc_.[[Writable]] is present and has value *true*, return *false*. 1. If _Desc_.[[Enumerable]] is present and has value *false*, return *false*. 1. If _Desc_.[[Configurable]] is present and has value *true*, return *false*. 1. If _Desc_.[[Value]] is present, return ! SameValue(_Desc_.[[Value]], _current_). 1. Return *false*.

[[HasProperty]] ( _P_ )

When the [[HasProperty]] internal method of a Record exotic object _R_ is called with property key _P_, the following steps are taken:

1. Assert: IsPropertyKey(_P_) is *true*. 1. If Type(_P_) is a Symbol, return *false*. 1. Return ! RecordHasProperty(_R_, _P_).

[[Get]] ( _P_, _Receiver_ )

When the [[Get]] internal method of a Record exotic object _R_ is called with property key _P_ and ECMAScript language value _Receiver_, the following steps are taken:

1. Assert: IsPropertyKey(_P_) is *true*. 1. If Type(_P_) is a Symbol, return *undefined*. 1. Return ! RecordGet( _R_, _P_ ).

[[Set]] ( _P_, _Receiver_ )

When the [[Set]] internal method of a Record exotic object _R_ is called with Property Key _P_ and ECMAScript language value _Receiver_, the following steps are taken:

1. Return *false*.

[[Delete]] ( _P_ )

When the [[Delete]] internal method of a Record exotic object _R_ is called with property key _P_, the following steps are taken:

1. Assert: IsPropertyKey(_P_) is *true*. 1. If Type(_P_) is a Symbol, return *true*. 1. Let _hasProperty_ ? RecordHasProperty( _R_, _P_ ). 1. Return *true* if _hasProperty_ is *false*. 1. Return *false*.

[[OwnPropertyKeys]] ( )

When the [[OwnPropertyKeys]] internal method of a Record exotic object _R_ is called, the following steps are taken:

1. Let _keys_ be a new empty List. 1. Let _rec_ be _R_.[[RecordData]]. 1. Assert: Type(_rec_) is Record. 1. Let _len_ be the length of _rec_. 1. For each _entry_ of _rec_.[[Fields]], do 1. Let _key_ be _entry_.[[Key]] 1. Assert: Type(_key_) is String 1. Append _key_ to _keys_. 1. Return _keys_

RecordCreate ( _value_ )

This abstract operation RecordCreate takes a _value_ argument. It is used to specify the creation of new Record exotic objects. It performs the following steps when called:

1. Assert: Type(_value_) is a Record. 1. Assert: Type(_value_.[[Fields]]) is a List. 1. Let _R_ be ! MakeBasicObject(« [[RecordData]], [[Prototype]] »). 1. Set _R_'s essential internal methods to the definitions specified in . 1. Set _R_.[[Prototype]] to *null*. 1. Set _R_.[[RecordData]] to _value_. 1. Return _R_

RecordHasProperty ( _R_, _P_ )

This abstract operation RecordHasProperty takes arguments _R_ and _P_. It performs the following steps when called:

1. Assert: IsPropertyKey(_P_) is *true*. 1. If Type(_P_) is a Symbol, return *false*. 1. Let _rec_ be _R_.[[RecordData]]. 1. Assert Type(_rec_) is a Record. 1. For each _entry_ of _rec_.[[Fields]], do 1. Let _key_ be _entry_.[[Key]] 1. If ! SameValue(_key_, _P_), return *true*. 1. Return *false*.

RecordGet ( _R_, _P_ )

This abstract operation RecordGet takes arguments _R_ and _P_. It performs the following steps when called:

1. Assert: _R_ is an Object that has [[RecordData]] internal slot. 1. Assert: Type(_P_) is String. 1. Let _rec_ be _R_.[[RecordData]]. 1. Assert Type(_rec_) is Record. 1. Let _len_ be the length of _rec_. 1. For each _entry_ of _rec_.[[Fields]], do: 1. Let _key_ be _entry_.[[Key]]. 1. Let _val_ be _entry_.[[Value]]. 1. Assert: Type(_key_) is String. 1. Assert: Type(_val_) is not Object. 1. Return _val_.

Tuple Exotic Objects

A Tuple object is an exotic object that encapsuates a Tuple value and exposes virtual integer-indexed data properties corresponding to the individual entries set on the underlying Tuple value. All keys properties are non-writable and non-configurable.

An object is a Tuple exotic object (or simply, a Record object) if its following internal methods use the following implementations.

Tuple exotic objects have the same internal slots as ordinary objects. They also have a [[TupleData]] internal slot.

[[SetPrototypeOf]] ( _V_ )

When the [[SetPrototypeOf]] internal method of a Tuple exotic object _T_ is called with argument _V_, the following steps are taken:

1. Return ! SetImmutablePrototype(_T, _V_).

[[IsExtensible]] ()

When the [[IsExtensible]] internal method of a Tuple exotic object _T_ is called, the following steps are taken:

1. Return *false*.

[[GetOwnProperty]] ( _P_ )

When the [[GetOwnProperty]] internal method of a Tuple exotic object _T_ is called with property key _P_, the following steps are taken:

1. Assert: IsPropertyKey(_P_) is *true*. 1. If Type(_P_) is String, then 1. Let _numericIndex_ be ! CanonicalNumericIndexString(_P_). 1. If _numericIndex_ is not *undefined*, then 1. Let _value_ be ? TupleGet(_T_, _numericIndex_) 1. If _value_ is *undefined*, return *undefined*. 1. Return the PropertyDescriptor { [[Value]]: _value_, [[Writable]]: *false*, [[Enumerable]]: *true*, [[Configurable]]: *false* }. 1. Return *undefined*

[[HasProperty]] ( _P_ )

When the [[HasProperty]] internal method of a Tuple exotic object _T_ is called with property key _P_, the following steps are taken:

1. Assert: IsPropertyKey(_P_) is *true*. 1. If Type(_P_) is String, then 1. Let _numericIndex_ be ! CanonicalNumericIndexString(_P_). 1. If _numericIndex_ is not *undefined*, then 1. Return ? IsValidTupleIndex(_T_, _numericIndex_) 1. Return *false*

[[DefineOwnProperty]] ( _P_, _Desc_ )

When the [[DefineOwnProperty]] internal method of a Tuple exotic object _T_ is called with property key _P_, and Property Descriptor _Desc_ the following steps are taken:

1. Assert: IsPropertyKey(_P_) is *true*. 1. If Type(_P_) is String, then 1. Let _numericIndex_ be ! CanonicalNumericIndexString(_P_). 1. If _numericIndex_ is not *undefined*, then 1. Let _current_ be ? TupleGet(_T_, _numericIndex_) 1. If _current_ is *undefined* return *false*. 1. If IsAccessorDescriptor(_Desc_) is *true*, return *false*. 1. If _Desc_.[[Writable]] is present and has value *true*, return *false*. 1. If _Desc_.[[Enumerable]] is present and has value *false*, return *false*. 1. If _Desc_.[[Configurable]] is present and has value *true*, return *false*. 1. If _Desc_.[[Value]] is present, return ! SameValue(_Desc_.[[Value]], _current_). 1. Return *false*.

[[Get]] ( _P_, _Receiver_ )

When the [[Get]] internal method of a Tuple exotic object _T_ is called with property key _P_ and ECMAScript language value _Receiver_, the following steps are taken:

1. If Type(_P_) is String, then 1. Let _numericIndex_ be ! CanonicalNumericIndexString(_P_). 1. If _numericIndex_ is not *undefined*, then 1. Return ? TupleGet(_T_, _numericIndex_) 1. Return *undefined*

[[Set]] ( _P_, _Receiver_ )

When the [[Set]] internal method of a Tuple exotic object _T_ is called with property key _P_ and ECMAScript language value _Receiver_, the following steps are taken:

1. Return *false*.

[[OwnPropertyKeys]] ( )

When the [[OwnPropertyKeys]] internal method of a Record exotic object _R_ is called, the following steps are taken:

1. Let _keys_ be a new empty List. 1. Assert _T_ is Tuple exotic object. 1. Let _len_ be _T_.[[TupleLength]]. 1. For each integer _i_ starting with 0 such that _i_ < _len_ in ascending order, do 1. Add ! ToString(_i_) to _keys_. 1. Return _keys_

TupleCreate ( _value_ )

This abstract operation TupleCreate takes a _value_ argument. It is used to specify the creation of new Tuple exotic objects. It performs the following steps when called:

1. Assert: Type(_value_) is Tuple. 1. Assert: Type(_value_.[[Sequence]]) is List. 1. Let _T_ be ! MakeBasicObject(« [[TupleData]], [[Prototype]] »). 1. Let _T_.[[TupleLength]] be _value_.[[Sequence]].length 1. Set _T_'s essential internal methods to the definitions specified in . 1. Set _T_.[[Prototype]] to *null*. 1. Set _T_.[[TupleData]] to _value_. 1. Return _T_

IsValidTupleIndex ( _T_, _numericIndex_ )

This abstract operation IsValidTupleIndex takes arguments _T_ and _numericIndex_. It performs the following steps when called:

1. Assert: _T_ is an Object that has [[TupleData]] internal slot. 1. Assert: Type(_numericIndex_) is an integer. 1. If _numericIndex_ is *-0*, return *false* 1. If _numericIndex_ < 0 or index >e; _T_.[[TupleLength]], return *false*. 1. Return *true*.

TupleGet ( _T_, _numericIndex_ )

This abstract operation TupleGet takes arguments _T_ and _numericIndex_. It performs the following steps when called:

1. Assert: _T_ is an Object that has [[TupleData]] internal slot. 1. Assert: Type(_numericIndex_) is an integer. 1. Let _tup_ be _T_.[[TupleData]]. 1. If ! IsValidTupleIndex(_numericIndex_) is *false*, return *undefined*. 1. Let _value_ be the element with index _numericIndex_ in _tup_.[[Sequence]] 1. Return _value_

Ordinary and Exotic Objects Behaviours

Built-in Exotic Object Internal Methods and Slots

Record Exotic Objects

A Record object is an exotic object that encapsuates a Record value and exposes virtual string-indexed data properties corresponding to the individual keys set on the underlying Record value. All keys properties are non-writable and non-configurable.

An object is a Record exotic object (or simply, a Record object) if its [[GetOwnPoperty]], [[DefineOwnProperty]], and [[OwnPropertyKeys]] internal methods.

Record exotic objects have the same internal slots as ordinary objects. They also have a [[RecordData]] internal slot.

[[GetOwnProperty]] ( _P_ )

When the [[GetOwnProperty]] internal method of a Record exotic object _R_ is called with property key _P_, the following steps are taken:

1. Assert: IsPropertyKey(_P_) is *true*. 1. If Type(_P_) is a Symbol, throw a *TypeError* exception. 1. Return ! RecordGetOwnProperty(_R_, _P_).

[[DefineOwnProperty]] ( _P_, _Desc_ )

When the [[DefineOwnProperty]] internal method of a Record exotic object _R_ is called with property key _P_, and Property Descriptor _Desc_ the following steps are taken:

1. Assert: IsPropertyKey(_P_) is *true*. 1. Throw a *TypeError* exception.

[[OwnPropertyKeys]] ( )

When the [[OwnPropertyKeys]] internal method of a Record exotic object _R_ is called, the following steps are taken:

1. Let _keys_ be a new empty List. 1. Let _rec_ be _R_.[[RecordData]]. 1. Assert: Type(_rec_) is Record. 1. Let _len_ be the length of _rec_. 1. For each primitive data property _pdp_, do 1. Let _key_ be the String key value of _pdp_ 1. Assert: Type(_key_) is String 1. Add _key_ to _keys_ 1. Return _keys_

RecordGetOwnProperty ( _R_, _P_ )

This abstract operation RecordGetOwnProperty takes arguments _R_ and _P_. It performs the following steps when called:

1. Assert: _R_ is an Object that has [[RecordData]] internal slot. 1. Assert: Type(_P_) is String. 1. Let _rec_ be _R_.[[RecordData]] 1. Assert Type(rec) is Record 1. Let _len_ be the length of _rec_. 1. Let _resultPrim_ be *undefined* 1. For each primitive data property _pdp_, do 1. Let _key_ be the String key value of _pdp_ 1. Let _prim_ be the associated value of _pdp_ 1. Assert: Type(_key_) is String 1. Assert: Type(_prim_) is not Object 1. If _key_ == _P_, let _resultPrim_ be _prim_ 1. If _resultPrim_ is *undefined*, return *undefined* 1. Return the PropertyDescriptor { [[Value]]: _resultPrim_, [[Writable]]: *false*, [[Enumerable]]: *true*, [[Configurable]]: *false* }.
